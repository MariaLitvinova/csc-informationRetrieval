Индексатор
На вход индексатору подаётся два параметра: путь до папки с документами, которые требуется проиндексировать, и путь до MyStem.exe (вместе с .exe). Индексатор делает следующее:

- обрабатывает коллекцию документов
- создаёт папку index рядом с исполняемым файлом
- кладёт в эту папку файл с индексом

UPD: Если передать индексатору не три параметра, а четыре (пока что четвёртый может быть любым), то построится координатный индекс (в директории coordinateIndex, для каждого файла свой).
